# KitFreeMiNT

KitFreeMiNT est une distribution FreeMiNT basée sur pkgsrc pour l'émulateur ARAnyM

# Objectifs

- Distribution simple à générer et à mettre à jour
- pkgsrc : 29000 logiciels potentiellement disponibles
- Remettre au goût du jour le noyau FreeMiNT et la bibliothèque C MiNTlib
